# Signaleditor

Web app for editing Swiss railway signal data files (YAML). Built with SvelteKit, Svelte 5, and TypeScript. Deployed on Vercel with Blob storage.

## Getting Started

```bash
npm install
npm run dev
```

See [docs/SETUP.md](docs/SETUP.md) for Vercel deployment and environment variable configuration.
